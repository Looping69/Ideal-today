## 🔧 Remaining Manual Steps

### 1. Set Production Environment Variables
In Netlify/Vercel, configure:
```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
VITE_SITE_URL=https://idealtoday.com
VITE_GOOGLE_MAPS_KEY=your-key
VITE_YOCO_PUBLIC_KEY=pk_live_XXXXXX  # Use LIVE key
```

### 2. Set Supabase Edge Function Secrets
```bash
npx supabase secrets set ALLOWED_ORIGIN=https://idealtoday.com
```

### 3. Configure Auth Redirects in Supabase
Supabase Dashboard → Authentication → URL Configuration:
- Site URL: `https://idealtoday.com`
- Redirect URLs: `https://idealtoday.com/*`

### 4. Restrict Google Maps API Key
Google Cloud Console → APIs & Services → Credentials:
- Add HTTP referrer restrictions for `https://idealtoday.com/*`

### 5. Optional: Install Sentry
```bash
npm install @sentry/react
```
Then add `VITE_SENTRY_DSN` to environment variables.
