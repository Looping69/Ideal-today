# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-28

### Added
- Property listings with search and filtering
- Booking system with payment integration (Yoco)
- Host dashboard with listings management
- Admin panel for platform administration
- User rewards and gamification system
- Wishlist functionality
- Messaging system between guests and hosts
- Review and rating system
- Google Maps integration for property locations
- Multi-step property creation wizard
- Email notifications via Supabase Edge Functions

### Security
- Row Level Security (RLS) on all database tables
- Supabase Auth integration
- Secure payment processing via edge functions

### Infrastructure
- Supabase backend (Postgres + Auth + Storage)
- Vite + React frontend
- Netlify/Vercel deployment ready
- 35 database migrations
